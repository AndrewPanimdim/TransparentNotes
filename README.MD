 to run the transparentnotes, you'll have to install the necessities

      npm install electron-builder --save-dev 

once done, you can run the application in two ways. First, by typing

      npm start

in the terminal in visual studio code, and the second one is by building the application by typing

      npm run build

then once done, you can now search the file where you built the application and just click the transparentnotes.exe file then you are good to go
